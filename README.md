# 身份证归属地数据库

## 介绍
通过输入身份证号码前17位（输入18位也没问题），可计算出第18位校验码和性别、归属地信息。 只支持中国身份证

## 使用说明

下载.exe文件打开即可运行（最好是自己把源代码编译再运行）
